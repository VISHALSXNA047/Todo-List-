# Todo List App

A modern and responsive Todo List application built using React.js. Users can add, edit, delete, and manage tasks efficiently with a clean user interface.

## Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Complete all tasks at once
- Remove completed tasks
- Filter tasks (All, Completed, Uncompleted)
- Dark Mode / Light Mode
- Toast Notifications
- Fully Responsive Design

## Tech Stack

- React.js
- JavaScript (ES6+)
- CSS3
- React Toastify
